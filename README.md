## Local Setup

```bash
# Clone this repo
git clone https://github.com/gerardvermaak-pixel/Kite.git
cd Kite

# Open in VS Code
code .

# Then simply open index.html in Live Server or deploy to Vercel
```

## Features
- Beautiful dark-luxury design with deep greens & gold accents
- Instant add-to-cart with micro-animations
- Slide-in cart + multi-step checkout with progress bar
- Product quick-view modals
- Live search & smart filters (Flower / Pre-Rolls / Concentrates)
- Confetti celebration on successful order
- Fully responsive (mobile-first)
- Zero build step — pure HTML + Tailwind CDN + Vanilla JS

## Tech Stack
- Single `index.html` (self-contained)
- Tailwind CSS via CDN
- Font Awesome icons
- Vanilla JavaScript for cart, modals, filters, checkout
